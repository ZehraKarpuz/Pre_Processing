# Pre_Processing , OpenCV
Derin Öğrenme modelinin beyin BT görüntülerinden, inmeli bölgeyi daha başarılı tespit edebilmesi için görüntülere ön işleme çalışması yapılmıştır.
Scikit Learn kütüphanesinde bulunan imread() metoduyla görüntünün okunması sağlanır.
OpenCV kütüphanesinin createCLAHE(), bileteralFilter() gibi metodları kullanılarak beyin bt görüntülerine ön işleme uygulanmıştır.
Matplotlib kütüphanesiyle verilerin görüntülenmesi sağlanır.
